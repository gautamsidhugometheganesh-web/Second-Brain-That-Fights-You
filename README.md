# Second-Brain-That-Fights-You
An AI productivity system that challenges, contradicts, and interrogates you instead of politely agreeing.
Core Features (MVP)

We only need 3 features to launch:

1️⃣ Memory Engine

Store goals, notes, plans

Embed everything

Timestamp importance

2️⃣ Contradiction Detector

Examples:

Goal vs action mismatch

Repeated abandoned ideas

Notes that contradict each other

3️⃣ Daily / Weekly Interrogation

AI generates:

3 uncomfortable questions

1 forced choice

1 “finish or delete” decision
